# Wait Steps

Learn how to create and use wait steps to control the timing and flow of your journeys.

## Basic Usage

Create a simple wait step:

```javascript
oppla.journey('delayed-notification', {
  steps: [
    {
      type: 'wait',
      duration: 5000 // Wait for 5 seconds
    },
    {
      type: 'banner',
      content: 'Welcome to Oppla!'
    }
  ]
});
```

## Wait Types

### Time-Based Wait

```javascript
oppla.journey('timed-sequence', {
  steps: [
    {
      type: 'banner',
      content: 'Starting in 5 seconds...'
    },
    {
      type: 'wait',
      duration: 5000
    },
    {
      type: 'banner',
      content: 'Here we go!'
    }
  ]
});
```

### Event-Based Wait

```javascript
oppla.journey('event-wait', {
  steps: [
    {
      type: 'wait',
      event: 'userAction',
      timeout: 30000 // Optional timeout in milliseconds
    },
    {
      type: 'banner',
      content: 'Action completed!'
    }
  ]
});
```

### Condition-Based Wait

```javascript
oppla.journey('conditional-wait', {
  steps: [
    {
      type: 'wait',
      condition: function() {
        return document.querySelector('.feature-loaded') !== null;
      },
      checkInterval: 1000, // Check every second
      timeout: 30000 // Timeout after 30 seconds
    },
    {
      type: 'banner',
      content: 'Feature is ready!'
    }
  ]
});
```

## Advanced Usage

### Multiple Conditions

```javascript
oppla.journey('complex-wait', {
  steps: [
    {
      type: 'wait',
      conditions: [
        {
          type: 'event',
          name: 'userAction'
        },
        {
          type: 'time',
          duration: 5000
        }
      ],
      logic: 'OR' // Wait for either condition
    },
    {
      type: 'banner',
      content: 'Proceeding...'
    }
  ]
});
```

### Progress Tracking

```javascript
oppla.journey('progress-wait', {
  steps: [
    {
      type: 'wait',
      duration: 10000,
      showProgress: true,
      progressMessage: 'Loading...',
      onProgress: function(progress) {
        console.log(`Progress: ${progress}%`);
      }
    },
    {
      type: 'banner',
      content: 'Complete!'
    }
  ]
});
```

### Error Handling

```javascript
oppla.journey('error-handling', {
  steps: [
    {
      type: 'wait',
      condition: function() {
        return document.querySelector('.feature-loaded') !== null;
      },
      timeout: 30000,
      onTimeout: function() {
        console.error('Feature failed to load');
        // Handle timeout
      },
      onError: function(error) {
        console.error('Wait step error:', error);
        // Handle error
      }
    }
  ]
});
```

## Best Practices

1. **Set Reasonable Timeouts**: Always set appropriate timeout values
2. **Handle Errors**: Implement error handling for wait conditions
3. **Show Progress**: Use progress indicators for long waits
4. **Be Specific**: Use precise conditions for event-based waits
