# Analytics Integration

Learn how to integrate analytics with your Oppla journeys to track user engagement and measure success.

## Basic Tracking

Enable basic analytics for a journey:

```javascript
oppla.journey('welcome', {
  steps: [
    {
      type: 'banner',
      content: 'Welcome to Oppla!'
    }
  ],
  analytics: {
    enabled: true,
    trackViews: true,
    trackInteractions: true
  }
});
```

## Event Tracking

### Track Journey Events

```javascript
oppla.journey('onboarding', {
  steps: [
    {
      type: 'banner',
      content: 'Welcome to onboarding!'
    }
  ],
  analytics: {
    events: [
      {
        name: 'journey_started',
        properties: {
          journey_name: 'onboarding',
          step_count: 1
        }
      },
      {
        name: 'journey_completed',
        properties: {
          journey_name: 'onboarding',
          completion_time: 'duration'
        }
      }
    ]
  }
});
```

### Track Step Events

```javascript
oppla.journey('feature-tour', {
  steps: [
    {
      type: 'tooltip',
      content: 'Welcome to the feature tour!',
      analytics: {
        events: [
          {
            name: 'step_viewed',
            properties: {
              step_type: 'tooltip',
              step_index: 0
            }
          },
          {
            name: 'step_interaction',
            properties: {
              step_type: 'tooltip',
              step_index: 0,
              interaction_type: 'click'
            }
          }
        ]
      }
    }
  ]
});
```

## User Properties

### Track User Segments

```javascript
oppla.journey('segmented', {
  steps: [
    {
      type: 'banner',
      content: 'Segmented message'
    }
  ],
  analytics: {
    userProperties: {
      segment: 'premium',
      journey_status: 'active',
      last_interaction: 'timestamp'
    }
  }
});
```

### Track User Progress

```javascript
oppla.journey('progress-tracked', {
  steps: [
    {
      type: 'banner',
      content: 'Track your progress'
    }
  ],
  analytics: {
    progress: {
      track: true,
      properties: {
        current_step: 'index',
        total_steps: 'count',
        completion_percentage: 'calculated'
      }
    }
  }
});
```

## Advanced Analytics

### A/B Testing

```javascript
oppla.journey('ab-test', {
  steps: [
    {
      type: 'banner',
      content: 'A/B test message'
    }
  ],
  analytics: {
    abTest: {
      enabled: true,
      variant: 'A',
      metrics: [
        'view_count',
        'interaction_rate',
        'completion_rate'
      ]
    }
  }
});
```

### Custom Metrics

```javascript
oppla.journey('custom-metrics', {
  steps: [
    {
      type: 'banner',
      content: 'Track custom metrics'
    }
  ],
  analytics: {
    customMetrics: [
      {
        name: 'time_to_interact',
        type: 'duration',
        start: 'step_viewed',
        end: 'step_interaction'
      },
      {
        name: 'interaction_count',
        type: 'counter',
        event: 'step_interaction'
      }
    ]
  }
});
```

## Integration with External Analytics

### Google Analytics

```javascript
oppla.journey('ga-integration', {
  steps: [
    {
      type: 'banner',
      content: 'Google Analytics integration'
    }
  ],
  analytics: {
    integrations: {
      googleAnalytics: {
        enabled: true,
        events: [
          {
            name: 'journey_start',
            gaEvent: 'begin_checkout',
            properties: {
              journey_name: 'ga-integration'
            }
          }
        ]
      }
    }
  }
});
```

### Custom Analytics

```javascript
oppla.journey('custom-analytics', {
  steps: [
    {
      type: 'banner',
      content: 'Custom analytics integration'
    }
  ],
  analytics: {
    custom: {
      onEvent: function(event, properties) {
        // Send to custom analytics
        customAnalytics.track(event, properties);
      }
    }
  }
});
```

## Best Practices

1. **Track Key Events**: Monitor important journey milestones
2. **Use Meaningful Properties**: Include relevant context in events
3. **Test Analytics**: Verify tracking is working correctly
4. **Analyze Results**: Use data to improve journey effectiveness
