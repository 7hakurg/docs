# Journey Overview

Oppla's journey features help you create engaging, guided experiences for your users.

## What are Journeys?

Journeys are multi-step user experiences that guide users through your product:

- Onboarding flows
- Feature discovery tours
- Product walkthroughs
- Interactive tutorials

## Key Features

### Step Types

Create engaging experiences with different step types:

- **Tooltips**: Highlight specific UI elements
- **Modals**: Display important information
- **Banners**: Show announcements or notifications
- **Wait Steps**: Control flow timing
- **Fork Steps**: Create conditional paths

### Customization

Customize your journeys to match your brand:

- Colors and typography
- Animations and transitions
- Custom CSS
- Responsive design

## Journey Management

### Frequency Control

Control how often users see your journeys:

```javascript
oppla.journey('onboarding', {
  frequency: 'once',
  steps: [
    {
      type: 'tooltip',
      target: '#signup-button',
      content: 'Click here to get started!'
    }
  ]
});
```

### Segmentation

Target specific user segments:

```javascript
oppla.journey('feature-tour', {
  segment: {
    plan: 'premium',
    lastLogin: '>7d'
  },
  steps: [
    // Journey steps
  ]
});
```
